# Dotfiles #

## Dependencies ##

* [GIT](http://git-scm.com/)
* [SVN](https://subversion.apache.org/)

## Install ##

Run this:

```bash
git clone git@github.com:claudiosanches/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
git submodule init
git submodule update
./install
```

## Update ##

```bash
git submodule update --remote
```
