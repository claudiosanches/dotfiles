# Dotfiles #

## Dependencies ##

* [GIT](http://git-scm.com/)
* [SVN](https://subversion.apache.org/)

## Install ##

Run this:

```bash
git clone git@github.com:claudiosmweb/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install
```
